# PintuKode

PintuKode merupakan platform yang dibuat sebagai sarana pembelajaran yang berbentuk seperti online modul untuk memudahkan proses belajar mentee untuk mengenal dan memperdalam pengetahuan terkait Frontend.

Materi disusun mengikuti praktik terbaru ekosistem JavaScript (React 19, Next.js 16/App Router, React Native, TypeScript, dan tool modern).

## Documentation

- [Deployment](https://pintu-kode.vercel.app/)
- [Repository](https://github.com/Luxxn12/PintuKode)

## Tech Stack

**Client:** Next.js, TailwindCSS, Nextra, MDX, Million.js
