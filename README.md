# PintuKode

PintuKode merupakan platform yang dibuat sebagai sarana pembelajaran yang berbentuk seperti online modul untuk memudahkan proses belajar mentee untuk mengenal dan memperdalam pengetahuan terkait Frontend.

## Documentation

- [Deployment](https:/PintuKode.Luxxn12.com)
- [Repository](https://github.com/Luxxn12/PintuKode)

## Tech Stack

**Client:** Next.js, TailwindCSS, DaisyUi, MDX, Million.js

## Contributing

Kamu dapat ikut andil berkontribusi dalam project ini dengan cara:
